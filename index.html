Première partie

Le Pseudo-code

Qu'est-ce que le pseudo-code ?

  • 📝 Une manière simple de décrire des étapes.
  • ❌ Ce n'est pas un vrai langage de programmation.
  • ✅ Sert uniquement à représenter une logique.
Discours : Le pseudo-code, c'est comme une recette de cuisine. Pas de syntaxe compliquée, juste du bon sens pour dire à l'ordinateur quoi faire.

Les 2 types d'exercices

1. Trouver le résultat attendu

Déterminez la valeur finale affichée (Expected Output).

A ← 5; B ← 2;
Afficher A+B

Résultat : 7

2. Code manquant

Complétez l'instruction manquante.

Si A > 5 Alors
  [ ??? ]

Résultat : Afficher "Bravo"

Discours : Vous aurez principalement deux missions : soit jouer à l'ordinateur en devinant le résultat final, soit corriger/compléter le code à trous.

📦 Les Variables

Une boîte avec un nom, qui contient une valeur.

Age ← 25
Prenom ← "Ali"
Age ← Age + 1
// La boîte Age contient maintenant 26
Expliquer l'analogie de la boîte. La valeur à l'intérieur peut changer, d'où le nom "variable".

MOD
Modulo

Le Modulo est le reste d'une division.
Exemple : 5 MOD 2 = 1 (car 5 = 2*2 + 1)

Insistez sur le Modulo, c'est l'opération la moins connue des débutants.

🔀 Les Conditions

C'est un arbre de décision simple de la vie de tous les jours. L'ordinateur fait exactement pareil.

🔗 Opérateurs Logiques

Toutes les conditions doivent être vraies.

NON (NOT)

Inverse la condition (Vrai devient Faux).

🔁 Les Boucles

POUR / TANT QUE

Répéter une action plusieurs fois de manière automatique.

TANT QUE (soif == VRAI)
   Boire de l'eau
FIN TANT QUE
POUR i DE 1 À 3
   Boire de l'eau
FIN POUR
Animation mentale : on vérifie la condition, on execute l'action, et on boucle tant que la condition reste vraie.

🔳 Matrices de Raven

Cherchez la logique cachée dans les figures :

  • 🔹 Formes et Couleurs
  • 🔄 Rotations
  • 🪞 Symétries
  • ➕ Répétitions / Additions de formes
Expliquez que c'est un puzzle visuel. Il faut trouver la règle qui lie les images entre elles, que ce soit de gauche à droite ou de haut en bas.

🔢 Suites numériques

Identifiez la règle mathématique :

2 ➡️ 4 ➡️ 6 ➡️ 8 ➡️ ?
Logique : +2 à chaque fois
1 ➡️ 2 ➡️ 4 ➡️ 8 ➡️ ?
Logique : x2 à chaque fois

✅ Conseils pour réussir

  • 📖 Lire calmement chaque exercice.
  • 👁️ Observer tous les détails.

  • 🪜 Procéder étape par étape.
  • 🎲 Ne pas répondre au hasard.
Insistez sur la gestion du stress. C'est un test de logique, la précipitation est l'ennemi.

  • 🙈 Ignorer une information clé.
  • 🏃 Se précipiter pour finir vite.

  • 😰 Se laisser envahir par le stress.
